# Camera-based Person Re-identification
The official code for [A Multi-Background Normalization and Dynamic Meta Feature Mining Approach for Person Re-Identification](https:).
It implements the fundamental idea of our paper: 1. actively aligning and normalizing all cameras; 2. Dynamic margin mining and uniform convergence, finally focusing pedestrain dicrimination.
This code is based on an early version of [Cysu/open-reid](https://github.com/Cysu/open-reid) and [Rethinking the Distribution Gap of Person Re-identification with Camera-based Batch Normalization](https:).
